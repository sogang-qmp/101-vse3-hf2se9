# VSe3 TP/TAP Structure and Hf-doped Molecule Junction in CNTs

## Idea

Transition metal trichalcogenide (MX3)는 1D chain 구조를 가지며, TP (trigonal prismatic) 또는 TAP (trigonal anti-prismatic) 두 가지 배위 구조가 가능하다. Te 계열은 TAP, Nb는 TP로 알려져 있으나, **VSe3는 실험적 리포트가 없었다**. CNT 내부에서 합성된 VSe3 single chain의 구조를 atomic resolution STEM으로 identify하여 **TP 구조임을 최초 확인**하고, DFT 계산으로 TP vs TAP 안정성을 비교한다.

**핵심 질문:**
1. VSe3 single chain에서 TP vs TAP 중 어느 구조가 안정한가?
2. Se vs Te에 따른 TP/TAP 선호도 차이 (기존 JACS Te 데이터 참조)
3. Hf 도핑 molecule 구조가 CNT 내부에서 안정화되는가?

**확장:**
- Hf 도핑 → 1D molecule junction 관찰 + 구조 제안
- 궁극적 목표: 1D metal superlattice 구조 (concept proposal)
